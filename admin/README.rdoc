== Admin retiqueta

This is a small web app that provides an admin module for the retiqueta app.

# Usage

1) Make sure that you have the retiqueta api containers running. See https://github.com/rafael/retiqueta-api/ for more instruction in how to do that.
2) Run the admin container:

```
$ docker-compose up
```

3) Go to the following url in you docker-machine ip:

```
http://192.168.99.100:3002/admin
```
